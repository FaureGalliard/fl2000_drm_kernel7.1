# FL2000 USB Display Driver for Linux Kernel 6.x

Welcome to the official documentation for the FL2000 USB Display Driver, updated for Linux Kernel 6.x.

## Overview

This driver supports USB-to-HDMI adapters based on the Fresco Logic FL2000DX chip with the IT66121FN HDMI bridge. It has been modernized to work with Linux kernel 6.x and later.
